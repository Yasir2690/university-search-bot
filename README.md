> \\\*\\\*A complete web application to search, compare, and get detailed information about Indian universities\\\*\\\*



\## ✨ Features



\- 🔍 \*\*Smart Search\*\* - Search by name or aliases (LPU, IIT Bombay, VIT)

\- 🏷️ \*\*Category Filters\*\* - IITs, NITs, IIITs, Private Universities

\- 📊 \*\*Detailed Info\*\* - Fees, placements, cutoff ranks, admission process

\- 🎓 \*\*Course Search\*\* - Find universities by course

\- ⭐ \*\*Student Reviews\*\* - Read and add reviews

\- 📱 \*\*Responsive Design\*\* - Works on mobile, tablet, desktop
